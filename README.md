# Event

## MethodType

| type   | description                    |
|--------|--------------------------------|
| get    | Get method call (on demand)    |
| update | Update method call (scheduled) |

## EventType

| type | description                      |
|------|----------------------------------|
| hitA | Cache hit before synchronization |
| hitS | Cache hit after synchronization  |
| miss | Cache miss, updated value        |
